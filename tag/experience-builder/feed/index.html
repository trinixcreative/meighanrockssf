Every Community has a Guest User and Guest User Profile. A guest is someone that can see your community before they login - an Unauthenticated user. Each time a community or portal is created, Salesforce auto-creates a guest user profile and a site guest user record. No matter what, even if your community is authenticated, you have a guest user and the site guest user record is the only user record associated with the guest user profile.
